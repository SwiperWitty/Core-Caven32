@par Example Description 

This Demo provides a description of how to use the USB-FS-Device on the AT32.
The AT32 device is enumerated as an USB-FS-Device Joystick Mouse, that uses 
the native PC Host USB-FS-Device HID driver.

The demo run on AT-START board.
