@par Example Description 

This example provides a description of how to use the USB OTG Device peripheral 
on the AT32F415xx devices.
The AT32 device is enumerated as an USB Device keyboard, that uses the
native PC Host HID driver.

The Keyboard mounted on the AT-START boards is used to emulate the Keyboard directions.
