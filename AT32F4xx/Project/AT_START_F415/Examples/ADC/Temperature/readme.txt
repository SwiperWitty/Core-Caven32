@par Example Description

This example describes how to use the ADC1 and DMA to sample the temperature. 
Temperature sensor is connected to channel16. The ADCs clock is set to 18 MHz.
Note: When run in AT-START v1.0, please confirm power only supply by USB,
othwise, the interner temperature is not correct.
